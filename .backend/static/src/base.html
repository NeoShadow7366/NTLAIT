Inference Studio — Guide

Model Checkpoint / Refiner / VAE

Select the base diffusion model. Optionally chain a Refiner (e.g. SDXL-Refiner) for enhanced detail in second-pass. VAE overrides the baked image decoder — useful for fixing colour shifts.

LoRA Network

Add fine-tuning adapters stacked on the base model. Each slot has a weight slider — higher values apply the style more strongly.

Sampler Engine & Scheduler

Sampler controls the noise removal algorithm. Euler A is fast and creative; DPM++ 2M Karras produces crisp, consistent results. Scheduler shapes the noise curve — Karras is recommended for most use cases.

Steps, CFG Scale, Seed

Steps = how many refinement passes (20–30 is typical). CFG = how closely the model follows your prompt (7 is balanced, 12+ is strict). Seed = -1 for random, or fix a value to reproduce an image.

High-Res Fix

Generate at base resolution then upscale and re-sample for sharper detail. Latent upscalers are fast; ESRGAN/SwinIR produce higher fidelity. Denoise 0.3–0.6 adds detail without changing composition.

Image-to-Image

Upload a reference image to guide generation. Denoise Strength 1.0 = full re-generation; lower values preserve more of the original.

🎨 Advanced Canvas Features

🖌️ Inpainting

Click the 🖌️ Inpaint button on the canvas toolbar to enter inpaint mode. Paint a red mask over the area you want to regenerate, then click Generate.

Toolbar controls: Brush/Eraser toggle, Size slider, Undo/Redo, Invert mask, Clear mask.

Denoise: Low (0.3) = subtle fix; High (1.0) = full regeneration of masked area.

Blur: Feathers mask edges (8–20px recommended) to prevent hard seams.

Fill modes: Original (preserves pixels), Noise (VAEEncodeForInpaint), Fill (Telea color fill), Latent Noise (full denoise).

🎯 Inpaint Model Override

In the inpaint toolbar, use the Inpaint Model dropdown to select a dedicated inpainting checkpoint (e.g. realisticVision-inpainting). This enables InpaintModelConditioning — a specialized node that feeds the image + mask into the model's extra conditioning channels for significantly better boundary blending.

Leave as "Use Main Model" to inpaint with your currently selected checkpoint.

🪄 SAM Smart Masking

Click the 🪄 SAM button, then click objects in the image to auto-generate pixel-perfect masks. Use Apply to replace or Add to combine with existing mask. Invert selects everything except the clicked object.

Requires ComfyUI-Impact-Pack extension (one-click install available).

↕️ Outpainting

While in Inpaint mode, use the arrow buttons (↑↓←→) around the image to extend the canvas in any direction. Choose extension size with the 128 / 256 / 512 buttons. A mask is auto-applied to the new region — click Generate to fill it.

Tip: Extend gradually (128–256px at a time) for more coherent results. Describe the scene continuation in your prompt.

🗺️ Regional Prompting

Click the 🗺️ Regions button to create zones with independent prompts. Drag to position, resize from edges. Each zone has its own prompt, negative prompt, and strength slider.

Per-Zone LoRA & IP-Adapter: Each zone can have its own LoRA model and reference image for granular style control.

⚡ Addons & Extensions

😊 FaceDetailer

Automatically detects and enhances faces after generation. Toggle on in the addon card. Adjust denoise (0.3–0.5 for subtle enhancement), max faces, cycles, and feather settings.

Requires ComfyUI-Impact-Pack.

🔲 LayerDiffuse

Generates images with transparent backgrounds (alpha channel). Toggle on and select the method matching your model type: SD15 Transparent or SDXL Transparent.

Requires ComfyUI-layerdiffuse extension.

🖼️ IP-Adapter

Use a reference image to guide style and composition. Available per regional zone — upload a reference image in each zone's IP-Adapter section.

Requires ComfyUI_IPAdapter_plus extension.

🔧 Architecture & More

⚡ FLUX Pipeline

Select flux-dev or flux-schnell model type. FLUX requires separate UNET, CLIP-L, and T5-XXL model files. Uses Flux Guidance (default 3.5) instead of CFG scale. VAE defaults to ae.safetensors — do not use SD/SDXL VAEs.

🎬 WAN Video

Select wan-video-txt (text-to-video) or wan-video-img (image-to-video). Configure Frames, FPS, and Motion Strength. Requires a WAN UNET model and CLIP text encoder.

📦 Batch Generation

Set Batch Size (1–8) to generate multiple variations per run. Each image uses a different random seed. All results appear in the gallery strip.

Gallery

All generated images appear in the gallery strip. Click a thumbnail to view it. Drag a thumbnail to the output area or the Config column to fully restore all generation settings. Scroll the mouse wheel over the output image to zoom.

🧭 Canvas Navigation

Scroll wheel to zoom in/out. Click + drag to pan (disabled during inpaint/SAM/region modes). Before/After comparison slider toggles between current and previous generation. Drag column dividers to resize panels.
