Inference Studio — Guide

Model Checkpoint / Refiner / VAE

Select the base diffusion model. Optionally chain a Refiner (e.g. SDXL-Refiner) for enhanced detail in second-pass. VAE overrides the baked image decoder — useful for fixing colour shifts.

LoRA Network

Add fine-tuning adapters stacked on the base model. Each slot has a weight slider — higher values apply the style more strongly.

Sampler Engine & Scheduler

Sampler controls the noise removal algorithm. Euler A is fast and creative; DPM++ 2M Karras produces crisp, consistent results. Scheduler shapes the noise curve — Karras is recommended for most use cases.

Steps, CFG Scale, Seed

Steps = how many refinement passes (20–30 is typical). CFG = how closely the model follows your prompt (7 is balanced, 12+ is strict). Seed = -1 for random, or fix a value to reproduce an image.

High-Res Fix

Generate at base resolution then upscale and re-sample for sharper detail. Latent upscalers are fast; ESRGAN/SwinIR produce higher fidelity. Denoise 0.3–0.6 adds detail without changing composition.

Image-to-Image

Upload a reference image to guide generation. Denoise Strength 1.0 = full re-generation; lower values preserve more of the original.

Gallery

All generated images appear in the gallery strip. Click a thumbnail to view it. Drag a thumbnail to the output area or the Config column to fully restore all generation settings. Scroll the mouse wheel over the output image to zoom.

Column Resizing

Drag the vertical dividers between columns to resize them to your preference.
